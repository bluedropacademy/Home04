## 📋 הוראות שימוש ב-Wix

### כדי להטמיע את דף הנחיתה באתר Wix שלך:

1. **פתח את עורך Wix** באתר שלך
2. **הוסף אלמנט HTML** (HTML Settings)
3. **בחר באפשרות "Code"** (לא Website address)
4. **הדבק את הקוד הבא:**

```html
<iframe 
  src="https://home04-bluedrop.netlify.app"
  width="100%"
  height="1800px"
  frameborder="0"
  scrolling="auto"
  style="border: none;">
</iframe>
```

5. **התאם את הגובה** (height) לפי הצורך - אם דף הנחיתה ארוך יותר, הגדל את המספר
6. **שמור ופרסם** את האתר

## 🔄 עדכון התוכן בעתיד

אם תרצה לעדכן את דף הנחיתה בעתיד:

### אפשרות 1: עדכון ידני דרך GitHub
1. היכנס ל-GitHub: https://github.com/bluedropacademy/Home04
2. ערוך את הקבצים ישירות בממשק
3. Netlify יעדכן אוטומטית את האתר

### אפשרות 2: חיבור אוטומטי (מומלץ)
1. היכנס לפאנל Netlify: https://app.netlify.com/projects/home04-bluedrop
2. לך ל-"Site settings" → "Build & deploy"
3. חבר את הריפוזיטורי מ-GitHub
4. כל שינוי ב-GitHub יעדכן אוטומטית את האתר
