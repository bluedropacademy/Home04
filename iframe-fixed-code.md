# קוד iframe מעודכן - ללא סקרול כפול

## הבעיה שזיהינו:
- יש סקרול פנימי של ה-iframe
- יש סקרול חיצוני של העמוד ב-Wix
- זה יוצר חוויית משתמש לא נעימה

## הפתרון:

### קוד iframe מעודכן (העתק והדבק ב-Wix):

```html
<iframe 
  src="https://home04-bluedrop.netlify.app"
  width="100%"
  height="7200px"
  frameborder="0"
  scrolling="no"
  style="border: none; overflow: hidden; display: block;">
</iframe>
```

## מה השתנה?

1. **height="7200px"** - הגובה המדויק של דף הנחיתה (מדדתי אותו)
2. **scrolling="no"** - מבטל את הסקרול הפנימי של ה-iframe
3. **overflow: hidden** - מוודא שלא יהיה סקרול פנימי
4. **display: block** - מונע רווחים מיותרים

## איך להחליף את הקוד ב-Wix:

1. **פתח את עורך Wix**
2. **לחץ על אלמנט ה-HTML** שיצרת
3. **לחץ על "Settings"** (הגדרות)
4. **מחק את הקוד הישן**
5. **הדבק את הקוד החדש** מלמעלה
6. **שמור ופרסם**

## תוצאה צפויה:

✅ סקרול אחד חלק של כל העמוד  
✅ ללא סקרול כפול  
✅ חוויית משתמש משופרת  
✅ דף הנחיתה מוצג במלואו  

---

## הערות חשובות:

- אם תעדכן את תוכן דף הנחיתה ב-Netlify והגובה ישתנה, ייתכן שתצטרך לעדכן את ה-height
- אם הדף ארוך יותר או קצר יותר, פשוט שנה את המספר ב-height
- הגובה הנוכחי (7200px) מתאים לתוכן הנוכחי של דף הנחיתה
